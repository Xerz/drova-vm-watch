## Лаунчер для лаунчеров GTA RP 

![Скриншот лаунчера](screenshot.png)

Лаунчер для лаунчеров GTA5RP и Majestic, прописывающий пути к GTA V Legacy перед запуском. Удобно использовать для карточки GTA5RP в дровах (предварительно скомпилировав в exe)

### Как использовать

1. Прописываем пути к установленным GTA V в секции "Пути к установленной GTA 5 по платформам"
2. Прописываем пути к лаунчерам Majestic и GTA5RP в секции "Пути к лаунчерам"
3. Компилируем скрипт в exe (AutoHotkey v2)
4. Настраиваем карточку GTA5RP в дровах, указывая путь к скомпилированному exe
